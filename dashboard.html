Technical Stack

Core Development

Python – core language for back-end logic, and Flask – lightweight web framework.

Computer Vision & Automation

OpenCV – real-time image processing, MediaPipe – advanced hand gesture detection, and PyAutoGUI – input automation.

Frontend & Data Management

HTML & CSS – web design and layout, and SQLite – user authentication system.
